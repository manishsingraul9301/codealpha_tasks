# 🎮 Hangman Game (Python)

## 📌 Goal
Create a simple text-based Hangman game where the player guesses a hidden word one letter at a time.

---

## 🚀 Features
- 🎯 Random word selection from a predefined list
- 🔤 Letter-by-letter guessing
- ❌ Maximum 6 incorrect attempts
- 📟 Console-based interaction (no graphics)

---

## 🧠 Concepts Used
- `random` module  
- `while` loop  
- `if-else` conditions  
- Strings & Lists
