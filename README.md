# 🤖 Basic Chatbot (Python)

## 📌 Goal
Build a simple rule-based chatbot that responds to user inputs using predefined replies.

---

## 🚀 Features
- 💬 Responds to basic inputs like greetings and questions  
- 🔁 Continuous conversation using loop  
- 🧠 Rule-based logic (no AI)  
- 📟 Console-based interaction  

---

## 🧠 Concepts Used
- `if-elif` conditions  
- Functions  
- Loops (`while`)  
- Input / Output
